1. Provision the MM app in LDAP Provider following the steps described here: https://docs.mattermost.com/deployment/sso-ldap.html
1.1 Add test data to LDAP Provider: https://github.com/mattermost/mattermost-server/blob/master/tests/test-data.ldif

2. set the environment variables via config
2.1 for local docker
    copy ldap.locahhost.json to cypress.env.json in the same folder as cypress.json
2.2 for AWS server
    copy ldap.locahhost.json to cypress.env.json in the same folder as cypress.json

3. set the environment variables via CYPRESS env variables
3.1 for local docker (OpenLdap):
    export CYPRESS_ldapServer=localhost
    export CYPRESS_ldapPort=389
3.2 for AWS server (OpenLdap):
    export CYPRESS_ldapServer=ldap.e2e.dev.spinmint.com
    export CYPRESS_ldapPort=389
