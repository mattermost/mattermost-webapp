## Mention Badges

Mention Badges are used to show a numerical count of mentions that a user has received. It can be found in
the Channel Sidebar as well as the App Header Bar.

In cases where there is a number larger than 99 displayed in the badge, we limit the badge display to 99+

## Unread Badges

Unread Badges are smaller badges used to inform the user that there are unread messages in a particular
Channel or Team. Unread Badges do not display a count.

## Status Badges

Status Badges are used to indicate the status of a particular user and are usually paired with an avatar
or username
